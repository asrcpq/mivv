# mivv

PyQt5 simple image viewer.

## features

* More mouse control(zoom, scroll)

## usage

1. have `XDG_CONFIG_HOME` and `XDG_CACHE_HOME` set

2. have pyqt5 installed

3. `python3 src/mivv.py`

## problem

* click during image load trigger image switch after load
