# mivv

PyQt5 simple image viewer.

![demo](https://asrcpq.github.io/resources/2111/mivv_demo.gif)

## features

* async image load and caching

* free rotation

* annotate image(wip)

## usage

1. have pyqt5 installed

2. `python3 mivv --help`

see control.md for keybind
